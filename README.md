🌤️ Weather App



A web-based application that fetches and displays real-time weather data using the OpenWeatherMap API, with support for both city-based search and device geolocation.



🚀 Features

City-based weather search

Current location weather detection

Real-time temperature, weather condition, and humidity

Dynamic UI updates based on weather conditions

🛠️ Tech Stack

HTML

CSS

JavaScript (DOM Manipulation, Fetch API)

OpenWeatherMap API

📌 Key Highlights

Integrated external REST API for live data

Implemented Geolocation API for location-based results

Handled API responses and error states

Dynamic rendering of UI based on weather conditions

⚙️ How to Run

Open Weather.html

Enter a city name or use device location
